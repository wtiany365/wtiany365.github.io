需要转化一个对象的接口用于另一个对象时，实现Adapter对象。
```php
<?php
//源
class oldObject{
    function method(){
        echo 'method';
    }
}
//目标
interface Target{
    function newMethod();
}
//适配器
class objectAdapter implements Target{
    private $obj;
    function __construct(oldObject $obj){
        $this->obj = $obj;
    }
    function newMethod(){
        return $this->obj->method();
    }
}
$a = new objectAdapter(new oldObject);
$a->newMethod();

```
